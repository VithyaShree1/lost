# Lost & Found Smart Campus

## Project Description
This project helps students and staff report lost items and found items in a college campus.

## Features
- Student Login
- Student Registration
- Report Lost Item
- Report Found Item
- Search Items
- Admin Dashboard
- Contact Page
- About Page

## Technologies Used
- HTML
- CSS
- JavaScript
